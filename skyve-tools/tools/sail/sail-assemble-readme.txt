Copy linux/mac chromedriver to tools/sail/
Copy windows chromedriver.exe to tools/sail/
Copy contents of windows jre folder of at least Java 8 JRE to tools/sail/jre/
Run launch4j
Open tools/sail/sail.xml
Build sail.exe to tools.sail/
Run ant target sailAssemble in skyve-build.xml
Sail folder is in content folder ready to be zipped when required.
