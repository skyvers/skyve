To install the ultima theme (>= 7.0.0) on skyve.

1) Drop ultima-theme-7.0.0-jakarta.jar from Foundry project into WEB-INF/lib/
2) Drop ultima-layout/ folder from Foundry project into WEB-INF/resources/ and rename to layout/ 
3) Change router.UxUis to use "ultima" for the template name, "ultima" for the theme name 
	and a theme colour matching the theme colour if there is one eg "indigo"

 To install the ecuador theme (>= 7.0.0) on skyve.
 
1) Drop ecuador-theme-7.0.0-jakarta.jar from Foundry project into WEB-INF/lib/
2) Drop ecuador-layout/ folder from Foundry project into WEB-INF/resources/
3) Change router.UxUis to use "ecuador" for the template name, "ecuador" for the theme name
	and a theme colour matching the theme colour if there is one eg "indigo"
